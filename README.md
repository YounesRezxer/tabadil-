# Tabdil
تومان به ریال حروف و عدد


‏اپلیکیشن تبدیل تومان به ریال، یک ابزار سریع و کاربردی برای محاسبه دقیق مبالغ در دسته‌چک و امور مالی روزمره است.

‏فقط عدد تومان را وارد کنید تا در لحظه مقدار معادل آن به ریال، هم به صورت عددی و هم حروفی نمایش داده شود.

‏✔ تبدیل سریع و دقیق

‏✔ نمایش عددی و متنی (حروفی)

‏✔ مناسب برای دسته‌چک و حسابداری

‏✔ رابط کاربری ساده و روان

‏یک ابزار سبک و کاربردی برای همه کسانی که با اعداد مالی سروکار دارند.
